# Kfir is Here

Personal landing page and portfolio for **Kfir Amoyal** — Junior Fullstack Developer / DevOps-aware Software Engineer.

## Live site

This project is intended to be deployed on Vercel.

## Highlights

- Cinematic scroll-driven landing page
- Light/Dark mode
- Project showcase with GitHub repository links
- Apollo/Hermes AI Agent concept section
- Certifications section
- Responsive static HTML/CSS/JavaScript

## Projects featured

- GeoDish
- Attenglish
- Hebrew TTS & Contextual Translation
- WinFlow

## Local preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8088
```

Then visit:

```text
http://localhost:8088
```
